# minikube
commnad to check minikube state|| minikube status
command to initiate a minikube environment using docker|| minikube start --driver=docker
command to view minikube webpage|| minikube dashboard

# kubectl
command to list all deployments|| kubectl get deployments
command to start a deployment object called mydeploy running image myapp|| kubectl create deployment mydeploy --image=myapp
command to delete mydeploy deployment|| kubectl delete deployment mydeploy
command to list all pods|| kubectl get pods
