AWS responsability: security of the cloud
customer responsability: security in the cloud

what is AWS Shield Standard?: is a service to protect from DDOS attack
what is AWS Shield Advanced?: is a premium version 24/7 of Shield Standard
what is AWS WAF?: web application firewall is a service to filter specific requests based on rules
what is AWS Network Firewall?: is a service to protect your AWS VPC with a firewall that works from layer 3 to layer 7
what is AWS Firewall Manager?: is a service to manage all security rules on all AWS accounts in an organization
what is AWS KMS?: key management service is used to encrypt data
what is CloudHSM?: is a service to provision hardware security module to manage your own encryption
what is ACM?: AWS Certificate Manager is service to provide SSL and TLS certificates for HTTPS in-flight encryption
what is Secret Manager?: is a service aimed to store and manage secrets
what is AWS Artifact?: is a portal to access on demand compliance reports and AWS agreements
what is AWS GuardDuty?: is a ML cyber security protection to detect unusual activity within your infrastructure
what is AWS Inspector?: is a service to analyze vulnerabilities on ECR, Lambdas, EC2 and network reachability
what is AWS Config?: is a service for recording compliance of AWS resources
what is AWS Macie?: is a ML service to alert about sensitive PII
what is AWS security Hub?: central security tool to manage several AWS accounts and automate security checks
what is AWS Detective?: is a service to analyze, investigate and identify root cause of security issues, capturing event process
what is AWS Abuse?: mail to report AWS resources for abusive or illegal purposes
what is IAM Access Analyzer?: is a service to report AWS resources access outside a defined zone of trust

AWS Shield and AWS WAF similarities: both are used to protect from DDOS Attack
AWS Shield and AWS WAF differences: AWS Shield protects on layer 7 while AWS WAF protects on layer 3 and 4
services that can be managed with AWS Firewall Manager:
	WAF rules
	AWS Shield
	AWS Network Firewall
	VPC security groups

types of attacks on AWS:
	penetration testing on specific services are allowed
	anything that may be seen like an attack is prohibited
types of encryption on AWS:
	at rest: stored data
	in transit: while data is being moved
types of KMS keys:
what are customer managed keys?: created, managed and used by the customer
what are AWS managed keys?: created, managed and used on customers behalf
what are AWS owned keys?: collection of CMKs owned by a service that can't be view by you
what are CloudHSM keys?: keys generated by your provisioned hardware
what ACM integrates with?: ELB, CloudFront and API Gateway
extra capabilities for AWS Secrets Manager: automatic rotation of secrets
AWS GuardDuty use cases: CryptoCurrency attacks and EventBridge anomally alerts
