what are AWS Organizations?: manage multiple AWS accounts
what are SCP?: Service Control Policies whitelist or blacklist IAM actions at account level
what is AWS Control Tower: easy way to set up and govern a secure and compliant multi account AWS environment
what is AWS RAM?: Resources Access Manager shares resources that you own with other AWS accounts
what is AWS Service Catalog?: service to access and create admin authorized services using cloud formation templates
what is AWS Compute Optimizer?: ML service to reduce costs by recommending optimal AWS resources
what is AWS Cost Anomaly Detection?: ML service to detect unusual usage or cost
what is AWS Service Quotas?: service to notify near threshold billing or shutdown services when reached
what is AWS Trusted Advisor?: AWS account configuration recommendations

pay less by using more pricing model: volume based disscounts
pay less as AWS grows pricing model: get disscounts as AWS scales
free tier pricing model: for tests and learning
types savings plans: ec2, compute, machine learning
