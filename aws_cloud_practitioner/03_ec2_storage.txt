# services
what is ebs?;; elastic block store is a network storage attachable to ec2 instances
what is ebs multi-attach?;; a new service to attach an ebs to multiple instances
what are ami?;; amazon machine image
what are ebs snapshots?;; backups from ebs volumes
what is efs?;; elastic file system is a network file system to share with multiple instances
what is efs infrequent access?;; storage class cost optimize for infrequent accessed files
what is fsx?;; amazon third party file systems

# ebs
is ebs volume a global service?;; no, it's bound to an availability zone
can we mount an ebs volume to multiple instances at the same time?;; no, only one at a time
can we attach an ebs volume in us-east-1a to an ec2 instance in us-east-1b?;; no, only in same zone
can we attach multiple ebs volumes to an instance?;; yes, but only in the same zone
how to create ebs ephemeral volumes?;; using delete on termination field

# snapshots
ebs snapshots use cases?;; backup data or transfer to a different zone in case of failure
what is recommended to do before an ebs snapshot?;; detach the volume from any instance
where can be restore snapshots?;; at any zone region
what are ebs snapshot archive?;; snapshot feature that is cheaper for infrequent access data
what are recycle bin snapshots?;; rules to retain deleted snapshots in case of accidental deletion from 1 day to 1 year

# ami
why would somebody use ami?;; faster boot, add software and configurations
types of ami;;
	public ami
	own ami
	marketplace ami
what is ec2 image builder?;; a service to automate creation(build, validate, test) of vms or containers

# instance store
what is the difference between ebs and ec2 instance store?;; ebs is a network disk, while ec2 instance store is attached to instances
when to use ec2 instance store over ebs?;; get high io performance
drawbacks of ec2 instance store;; it is ephemeral storage, risk of data loss in failure

# efs
what is the main advantage of efs over ebs?;; can be connected to multiple instances in multiple zones
what is the drawback of efs?;; it is 3x more expensive than ebs
fsx options;;
	windows file server(for windows servers)
	lustre(linux cluster for high performance computing)
